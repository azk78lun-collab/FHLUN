CDN 优选 IP 加速

cdnym（回源域名）：你的域名，已解析到 VPS IP。Cloudflare 通过它找到你的服务器。
cfip（优选地址）：客户端实际连接的 Cloudflare 入口，填 IP 或域名。
简单理解：客户端 → cfip（CF入口）→ cdnym（你的域名）→ VPS。仅对 VMess WS / VLESS WS / VLESS XHTTP 生效，端口必须在 CF 橙云端口内。
